# Yelp API Python Scrapper

## Overview
This program demonstrates the capability of the Yelp API
by using the Search API to query for businesses by a search term and location,
and the Business API to query additional information about the top result
from the search query.

Please refer to [API documentation](https://www.yelp.com/developers/documentation/v3)
for more details.


## Steps to run

To install the dependencies, run:
`pip install -r requirements.txt`

Edit the code and input your Yelp API key
`API_KEY= ' '`

Run the code sample without specifying any arguments:
`python Yelp.py`
